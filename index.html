Política de Privacidade – JESFAG Entregas

Última atualização: 30 de setembro de 2025

Sua privacidade é fundamental para nós. Esta política detalha como o aplicativo JESFAG Entregas coleta, utiliza e protege suas informações pessoais e de uso.

1. Coleta de Informações

a) Informações de localização

Coletamos dados de localização em tempo real apenas quando o entregador está utilizando o app para realizar entregas.

  • A localização é compartilhada somente enquanto o app está em uso e necessário para a entrega.
  • O usuário é informado no momento do uso e deve permitir o compartilhamento de localização.
  • Não coletamos localização quando o app está fechado.
  • Os dados são usados exclusivamente para rastrear rotas, estimar tempo de entrega e garantir segurança logística.

b) Informações de autenticação

  • Nome
  • Email
  • ID do Google

c) Informações técnicas

  • Versão do sistema operacional
  • Modelo do aparelho
  • Logs de erro

2. Armazenamento de Dados

Os dados são armazenados com segurança no Firebase (Google Cloud), com autenticação e criptografia.

3. Notificações

O app pode enviar notificações de entregas e status. Podem ser desativadas nas configurações do dispositivo.

4. Compartilhamento de Dados

  • Quando exigido por lei
  • Para funcionamento do app (ex: Google, Firebase)

5. Segurança

  • HTTPS para comunicação segura
  • Autenticação Firebase
  • Controle de acesso por perfil

6. Direitos do Usuário

  • Solicitar exclusão de dados
  • Revogar acesso à conta Google
  • Desinstalar o app

Your privacy is very important to us. This policy explains how the JESFAG Entregas app collects, uses, and protects your personal and usage information.

1. Information Collection

a) Location Information

We collect real-time location data only when the delivery driver is using the app to perform deliveries.

  • Location is shared only while the app is in use and necessary for delivery.
  • The user is informed when using the app and must allow location sharing.
  • We do not collect location when the app is closed.
  • Data is used solely to track routes, estimate delivery time, and ensure logistics security.

b) Authentication Information

  • Name
  • Email
  • Google ID

c) Technical Information

  • Operating system version
  • Device model
  • Error logs

2. Data Storage

All information is securely stored on Firebase (Google Cloud) using authentication and encryption.

3. Notifications

The app may send delivery and status notifications. Users can disable them in device settings.

4. Data Sharing

  • As required by law
  • For app operation (e.g., Google, Firebase)

5. Security

  • Secure HTTPS communication
  • Firebase authentication
  • Access control by user profile

6. User Rights
